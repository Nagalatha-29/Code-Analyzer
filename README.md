# Code Analyzer Chatbot

This project is an AI-based Code Analyzer Chatbot.  
It helps users analyze and understand code snippets using a Large Language Model.
Users can paste their code into the application, and the chatbot will analyze it and explain possible errors, logic issues, and improvements.

## Features

- Paste any code snippet for analysis  
- Detect possible bugs or issues  
- Explain the code step by step  
- Suggest improvements and best practices  
- Simple web interface using Streamlit  

## Technologies Used

- Python  
- Streamlit  
- LangChain  
- LangGraph  
- Groq LLM  

## How It Works

1. The user pastes a code snippet in the app.  
2. The system sends the code to the AI model.  
3. The AI analyzes the code.  
4. The chatbot returns an explanation and suggestions.
